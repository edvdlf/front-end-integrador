3.2. Solicitar relatório

Após a criação de templates, é possível gerar relatórios personalizados acessando esta funcionalidade.

Nome do relatório

Defina um nome para a solicitação de relatório.
Topo

Tipos de documentos

Selecione o tipo de documento, pré-configurado, ou seja, que foi criado pelo administrador do sistema, através da funcionalidade "Tipos de Documentos"
Topo

Template

Selecione o template pré-configurado. Lembrando que este template precisa ser um template que está vinculado há ao tipo de documento selecionado no campo, tipos de documento.
Topo

Fonte de dados

Selecione a Fonte de dados(configuração Url) pré-configurada. Lembrando que esta fonte de dados, precisa ser uma configuração que está vinculada há ao tipo de documento selecionado no campo, tipos de documentos.
Topo

Campo descrição

Descreva no campo descrição um breve resumo desta solicitação de relatório.
Topo

Campo CNPJs

Neste campo você deverá adicionar os CNPJs que serão utilizados na busca de dados para geração do relatório. Lembrando que para usar este recurso você deve digitar o cnpj (somente números), e clicar enter. Repita esta ação para cada CNPJ, que deseja adicionar na lista de cnpjs que irão compor seu relatório.
Topo

Data inicial

Informe a data inicial, que servirá de base para a consulta dos dados. Lembrando, que o periodo entre a data inicial e a data final, não pode ser superior a um mês.
Topo

Data final

Informe a data final, que servirá de base para a consulta dos dados.
Topo
