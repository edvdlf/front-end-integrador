4.1.1. Tipos de documentos configurados

 
Disponível apenas para administradores do sistema, essa funcionalidade permite configurar os tipos de documentos previstos pelo TAXDOCS Monitor, como **NF-e, NFS-e e CT-e**. Cada tipo de documento possui suas colunas e layouts específicos, que servirão de base para a criação de templates de relatórios.

Este card, exibe o total de tipos de documentos cadastrados no sistema, estejam eles, ativos ou inativos.
