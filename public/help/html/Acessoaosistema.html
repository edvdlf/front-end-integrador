Acesso ao sistema

Tela de Login

A tela de login é o ponto inicial de acesso ao Integrador Taxdocs Avatax.
Nela, o usuário informa seu e-mail e senha para entrar no sistema de forma segura.


Durante a implantação do Integrador, um usuário administrador padrão é criado automaticamente no banco de dados.
Esse usuário tem acesso total à aplicação e é responsável por:

Realizar o primeiro acesso ao sistema

Configurar informações iniciais

Cadastrar novos usuários

Definir perfis e permissões de acesso

Após o primeiro login, recomenda-se que o administrador:

Altere sua própria senha, por segurança

Cadastre os demais usuários que irão operar o sistema

Distribua as permissões conforme o papel de cada pessoa

A tela também disponibiliza os recursos:

Lembrar login

Recuperar senha (caso o usuário esqueça a senha cadastrada)

Essa estrutura garante que o acesso ao sistema seja seguro, controlado e centralizado desde o primeiro uso.
